What to consider when choosing your MacBook Pro.

Apple M3 chip

  • Brings serious speed and capability to blaze through everyday activities, multitask across apps, and take on creative projects
  • Run multiple apps, speed through tasks
  • Effortlessly edit 4K video
  • Configure with up to 24GB unified memory
  • Supports one external display

Apple M3 Pro chip

  • Building on the M3 chip, M3 Pro provides even greater performance and capability for demanding workloads
  • Manipulate gigapixel panoramas, compile code, edit multiple streams of 8K video
  • Configure with up to 36GB unified memory
  • Supports up to two external displays

Apple M3 Max chip

  • M3 Max powers the most extreme workflows with even more CPU and GPU cores, unified memory, and an advanced Media Engine
  • Render complex scenes, develop transformer models with billions of parameters
  • 8K video and pro audio
  • Configure with up to 128GB unified memory
  • Supports up to four external displays

Unified Memory

  • More efficient than traditional RAM, unified memory is faster and more responsive
  • Run multiple apps at once where performance remains fast and responsive
  • Add memory to run more apps simultaneously for faster, more fluid multitasking
  • With M3 Max, MacBook Pro can be configured with up to 128GB of memory
